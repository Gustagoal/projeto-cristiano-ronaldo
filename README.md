🚀 Projeto: Rumo ao 1000º Gol do Cristiano Ronaldo

Aplicação interativa desenvolvida em Python + Streamlit para análise e projeção da carreira de Cristiano Ronaldo, com foco na meta de 1000 gols.

🔧 Tecnologias
Python
Streamlit
Pandas/Openpyxl


📊 Funcionalidades
Visualização de gols por temporada
Cálculo de média de gols
Projeção do milésimo gol
Simulação de cenários


📁 Fonte de dados
ESPN


🎯 Objetivo
Demonstrar habilidades em:
Análise de dados
Visualização interativa
Desenvolvimento de dashboards
